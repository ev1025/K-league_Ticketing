# K-LEAGUE 인천 유나이티드 예매분석

**(데이터제공 기업과의 비밀유지계약체결로 일부 데이터가 비공개처리 될 수 있습니다.)**

### 1. 비지니스문제

**K-LEAGUE**
- 한국프로축구연맹 산하의 대한민국 프로축구리그입니다. 1983년 슈퍼리그라는 명칭으로 출범하였으며, 현재까지 AFC 챔피언스 리그 우승 횟수가 가장 많은 리그이기도 합니다.

**인천 유나이티드**
- 2003년에 창단 된 인천광역시의 K리그1 소속 프로축구단입니다.
- 부족한 재정과 잦은 선수 이탈로 시즌 초반에는 주로 하위권에 머물지만 승강제 도입 이후 한 번도 강등하지 않은 유일한 팀으로 **생존왕**이라는 별명을 가지고 있으며 하반기로 갈수록 팬들의 응원이 뜨겁습니다.

**비즈니스 문제**
- 코로나19로 인해 제한되었던 공연, 스포츠 관람이 완전 개방되면서 고객의 수요가 다시 늘어나는 추세입니다.
- K-LEAGUE 특성상 신규고객의 진입장벽이 높습니다. 하지만 작년 월드컵의 뜨거운 열기로 K-League에도 새로운 고객층의 관심이 많아진 상황입니다.

**프로젝트 목표**
- 2022년 인천유나이티드 경기장의 예매 데이터를 바탕으로 예매패턴을 분석하고 적절한 구매유도와 기존 고객의 리텐션 향상을 목표로 합니다.

**가설**
- 가설 1 : 게임 순위나 직전 경기결과가 예매에 영향을 미칠 것이다.
- 가설 2 : 고객세분화를 통해 적절한 구매유도 전략을 제시하고 예매를 예측할 수 있다.

### 2. 데이터 소개

**데이터**
- membership.csv : 고객아이디, 생년, 우편번호, 좋아하는 선수
- reserve_ticket.csv : 고객아이디, 예매번호, 티켓번호, 예매,취소시간, 경기날짜, 예매날짜
- d95e264436cf42de.xlsx : 경기 일자별 인천유나이티드 순위
- schedule2022120221019212245.xml : 경기 일자별 관객 수

**데이터 전처리**
- xml파일 디코딩중 오류가 발생하여 (디코딩->인코딩->디코딩)하여 문제 해결
- zipcode와 job에서 같은 수의 결측치가 발견되어 zipcode는 0, job은 기타로 변경
- 스카이라운지구역은 열번호가 없어서 결측치가 발생, sky로 변경
- 각 숫자데이터, 날짜데이터의 타입을 변경
